SAP UI5 Web Components is a set of reusable UI elements to help build enterprise-grade applications. These components deliver SAP Fiori user experience in a lightweight framework-agnostic way. They implement the latest SAP Fiori design guidelines and are fully accessible. The components can be used with any JavaScript framework — or no framework at all — making them perfect for a wide range of web applications.
